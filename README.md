# Spec Kit Templates

A curated library of reusable project templates for specification-driven software development.

> Status: foundation phase. The catalog structure is being defined and no complete template release is claimed yet.

## About

This repository will contain structured project starters with goals, requirements, plans, tasks, examples, and acceptance checklists. Templates are intended to be reviewed and adapted rather than copied without changes.

## Planned catalog

- Web application templates
- Desktop application templates
- Service and API templates
- Data-processing templates
- Automation templates
- Open-source project templates

## Template requirements

Every published template should include metadata, intended users, scope, non-goals, requirements, edge cases, test expectations, and a version history.

## Roadmap

1. Define the metadata schema.
2. Publish three complete reference templates.
3. Add structure validation.
4. Expand the reviewed catalog.
5. Publish versioned releases.

This is an independent community project and is not an official GitHub product.
