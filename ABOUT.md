# About Spec Kit Templates

## Suggested GitHub About

Curated specification-driven project templates with requirements, plans, tasks, examples, and acceptance checklists.

## Suggested topics

`specifications`, `project-templates`, `developer-tools`, `product-planning`, `documentation`, `open-source`

## Mission

The repository will provide complete, reviewable project starters instead of isolated prompts. Each template should help users move from a project idea to clear requirements, implementation planning, task breakdown, and measurable acceptance criteria.

## Intended users

Developers, product owners, educators, maintainers, and small teams using structured planning workflows.

## Scope

Template metadata, specifications, plans, tasks, examples, validation rules, review guidance, and versioned catalog releases.

## Current status

Foundation phase. The catalog schema and acceptance rules are not yet finalized.

## Maintenance

Template changes should include version notes. Catalog counts and compatibility statements must reflect committed, reviewed content only.
